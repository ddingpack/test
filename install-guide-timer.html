Documentation

띵팩 타이머 설치 가이드

현재 단계에서는 마인크래프트 모드 적용 기본 방법을 기준으로, 초보자도 바로 따라 할 수 있는 기본 설치 흐름을 먼저 정리했습니다. 추후 전용 상세 설명이 추가될 수 있도록 카드형 구조로 구성했습니다.

Step 1

Fabric 환경 준비

사용 중인 마인크래프트 버전에 맞는 Fabric 로더 환경을 준비합니다. 이미 Fabric 기반으로 플레이 중이라면 다음 단계로 넘어가면 됩니다.

Step 2

mods 폴더에 모드 파일 넣기

배포된 띵팩 타이머 모드 파일을 게임 디렉터리의 mods 폴더 안에 넣습니다. 다른 모드와 함께 사용할 경우 버전 호환도 같이 확인합니다.

Step 3

게임 실행 후 적용 확인

게임 실행 후 모드 목록 또는 실제 HUD/기능이 정상적으로 보이는지 확인합니다. 실행 오류가 있으면 모드 파일과 로더 버전을 다시 점검합니다.

Checklist

빠른 체크리스트

버전 호환 확인
Fabric 로더와 모드 파일 버전이 현재 게임 버전과 맞는지 확인합니다.
mods 폴더 위치 확인
모드 파일이 잘못된 폴더가 아니라 실제 mods 폴더에 들어가 있는지 확인합니다.
모드 충돌 점검
기존 설치된 HUD/타이머류 모드와 기능이 겹치지 않는지 점검합니다.
Future Update

추후 추가될 내용

전용 기능 설명
띵팩 타이머의 실제 UI와 사용 방법 안내가 추가될 예정입니다.
상세 호환 버전
배포가 시작되면 지원 버전과 주의사항이 더 구체적으로 정리됩니다.
