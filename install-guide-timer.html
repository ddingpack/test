띵팩 타이머 설치 가이드

띵팩 타이머는 띵타이쿤 온라인에서 곡나가와 광석 골렘 쿨타임을 더 편하게 확인할 수 있도록 도와주는 클라이언트 전용 모드입니다.

Fabric 1.21.4 클라이언트 전용
준비 단계

1. 사용 전 준비

  • Fabric 1.21.4 환경에서 사용해 주세요.
  • 다운로드한 모드 JAR 파일을 .minecraft/mods 폴더에 넣어 주세요.
  • 게임 실행 후 조작키 설정에서 띵팩 타이머 설정 열기 키를 확인할 수 있습니다.
설정창

2. 설정창 열기

  • 설정한 키를 누르면 띵팩 타이머 설정창이 열립니다.
  • 곡나가 레벨 / HUD 표시 / 광석 골렘 쿨타임 표시
  • 쿨타임 종료 시 표시 / HUD 위치 편집 / 타이머 초기화

3. HUD 위치 및 크기 조절

HUD 위치 편집을 누르면 HUD 편집 화면으로 들어갑니다. 편집 화면에서는 마우스로 HUD를 직접 드래그해서 위치를 옮길 수 있습니다.

참고 사항

5. 참고 사항

  • 본 모드는 클라이언트 전용입니다.
  • 게임 내 채팅/알림 문구를 감지해서 작동합니다.
  • 서버 문구가 변경되면 일부 기능이 정상 작동하지 않을 수 있습니다.
  • 다른 HUD 모드와 함께 사용할 경우 위치가 겹칠 수 있습니다.

초보자용 기본 흐름

  1. Fabric 1.21.4 환경 준비
  2. 다운로드한 모드 파일을 mods 폴더에 넣기
  3. 게임 실행 후 설정창 열기 키 확인
  4. HUD 표시와 알림 항목 점검
  5. 게임 내 문구가 정상 감지되는지 확인
버전 / 충돌

확인하면 좋은 항목

  • 사용 중인 마인크래프트 버전과 모드 버전이 맞는지 확인해 주세요.
  • 다른 HUD 모드가 화면 위치를 덮고 있지 않은지 확인해 주세요.
  • 문구 감지가 되지 않으면 서버 문구 변경 여부를 먼저 점검해 주세요.
