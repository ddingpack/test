Client Mod Guide

띵팩 타이머 설치 가이드

띵팩 타이머는 띵타이쿤 온라인에서 곡나가와 광석 골렘 쿨타임을 더 편하게 확인할 수 있도록 도와주는 클라이언트 전용 모드입니다.

Fabric 1.21.4 Client Mod

모드 다운로드

띵팩 타이머는 현재 바로 다운로드할 수 있습니다. 설치 전 Fabric 1.21.4 환경과 mods 폴더 위치를 먼저 확인해 주세요.

배포 중
다운로드

사용 전 준비

Fabric 1.21.4 환경에서 사용해 주세요. 다운로드한 모드 JAR 파일을 .minecraft/mods 폴더에 넣어 주세요.

Step 01

게임 실행 후 조작키 설정에서 띵팩 타이머 설정 열기 키를 확인할 수 있습니다.

설정창 열기

HUD 위치 및 크기 조절

HUD 위치 편집을 누르면 HUD 편집 화면으로 들어갑니다. 편집 화면에서는 마우스로 HUD를 직접 드래그해서 위치를 옮길 수 있습니다.
