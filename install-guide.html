Step Flow

설치 순서

원본 텍스트를 단계형 카드로 재구성해, 한 번에 따라 하기 쉬운 흐름으로 정리했습니다.

Step 01

Server Pack Unlocker를 먼저 설치합니다

개인 리소스팩을 서버 리소스팩 위에 배치하기 위한 선행 준비 단계입니다.

Step 02

Fabric API 환경을 준비합니다

본 리소스팩은 Fabric API 환경 기준으로만 안내합니다.

Step 03

Continuity를 함께 설치합니다

발광 텍스처와 연결 텍스처 표현을 위해 권장됩니다.

Step 04

리소스팩 배치 순서를 맞춥니다

특수광물 리소스팩이 띵타이쿤 온라인 공식 리소스팩보다 위에 오도록 배치해야 합니다.

위 · 특수광물 리소스팩 아래 · 띵타이쿤 온라인 공식 리소스팩
Step 05

Continuity의 빛나는 텍스쳐 옵션을 켜짐으로 설정합니다

해당 옵션이 꺼져 있으면 발광 효과가 정상 적용되지 않을 수 있습니다.

DDING PACK

띵타이쿤 플레이어를 위한 비공식 리소스팩 배포 페이지

DDING PACK은 띵타이쿤 유저를 위해 제작된 비공식 리소스팩 사이트이며, 공식 운영진과 무관합니다.

사이트 내 사용된 게임 관련 명칭 및 리소스의 출처와 권리는 “띵타이쿤 온라인”에 있습니다.
