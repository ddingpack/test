Installation Hub

설치할 자료를 먼저 선택해 주세요

설치 가이드를 자료별로 나눠, 리소스팩과 모드를 혼동하지 않고 필요한 안내만 바로 확인할 수 있도록 정리했습니다.

Resource Pack

Client Mod

Guide Note

자료별 설치 흐름

리소스팩은 적용 순서와 필수 모드 확인이 중요하고, 모드는 Fabric 환경과 mods 폴더 적용 여부가 중요합니다.

Quick Check

게임 버전, 자료 종류, 설치 위치를 먼저 확인하면 설치 실패를 크게 줄일 수 있습니다.

Reference
