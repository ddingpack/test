특수광물 발광 + 테두리 리소스팩 설치 가이드

기존 설치 가이드 내용을 기준으로, 리소스팩 설치 순서와 필수 모드, 적용 순서, 적용 예시, 자주 생기는 문제를 더 명확하게 정리했습니다.

Fabric API 기준 Continuity 권장
Step 1

필수 모드 확인

  • Server Pack Unlocker 사용이 필요합니다.
  • Continuity 사용이 가능해야 합니다.
  • 빛나는 텍스처 옵션을 켜짐으로 설정해야 발광 표현이 정상적으로 보입니다.
Step 2

리소스팩 적용 순서

  • 특수광물 발광 + 테두리 리소스팩을 추가합니다.
  • 띵타이쿤 온라인 공식 리소스팩보다 위에 배치합니다.
  • 적용 후 게임에 접속해 표시 상태를 확인합니다.
Preview

적용 예시

이미지를 클릭하면 더 크게 볼 수 있습니다.

Checklist

빠른 점검

  • 리소스팩 우선순서가 공식 리소스팩보다 위인지 확인
  • Continuity가 설치되어 있는지 확인
  • 빛나는 텍스처 옵션이 켜져 있는지 확인
  • Server Pack Unlocker가 정상 동작하는지 확인
Issues

자주 생기는 문제와 해결법

  • 적용이 안 되면 리소스팩 우선순서를 먼저 확인해 주세요.
  • 발광이 안 보이면 Continuity와 설정을 점검해 주세요.
  • 공식 리소스팩 위에 위치해야 정상 적용됩니다.
