Documentation

특수광물 발광 + 테두리 리소스팩 설치 가이드

기존 설치 가이드 내용을 유지하면서, 설치 순서와 필수 모드, 적용 예시, 문제 해결 정보를 카드형 구조로 다시 정리했습니다.

Step 1

필수 모드 확인

  • • Server Pack Unlocker 설치
  • • Continuity 사용 가능 환경 확인
  • • 빛나는 텍스쳐 옵션을 켜짐으로 변경
Step 2

리소스팩 적용 순서

  • • 특수광물 리소스팩 추가
  • • 공식 리소스팩보다 위에 배치
  • • 적용 후 게임 재확인
Check

빠른 점검 체크리스트

  • • 발광이 보이지 않으면 Continuity 확인
  • • 적용 안 되면 우선순서 확인
  • • 공식 리소스팩 위에 있는지 확인
Preview

적용 예시 이미지

이미지를 클릭하면 더 크게 볼 수 있습니다.
Order

적용 순서 요약

1. Server Pack Unlocker 설치
공식 리소스팩 위에 특수광물 리소스팩을 올리기 위해 필요합니다.
2. 특수광물 리소스팩이 위
공식 리소스팩보다 위에 위치해야 정상 적용됩니다.
3. Continuity 설정 확인
빛나는 텍스쳐 옵션이 켜짐이어야 발광 표현이 보입니다.
Troubleshooting

자주 생기는 문제

리소스팩이 적용되지 않아요
리소스팩 순서가 공식 리소스팩보다 위에 있는지 먼저 확인해 주세요.
발광 효과가 보이지 않아요
Continuity와 빛나는 텍스쳐 설정을 다시 점검해 주세요.
