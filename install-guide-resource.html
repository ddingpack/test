띵팩 리소스팩 설치 가이드

Fabric API 환경에서 사용하는 띵팩 리소스팩 설치 순서와 필수 모드, 리소스팩 배치, Continuity 옵션 설정을 단계별로 안내합니다.

01

필수 모드 준비

Fabric 환경에서 필요한 모드와 확장 모드를 먼저 확인해 주세요.

settings_suggest
check_circleFabric API 환경 준비
check_circleContinuity 설치
check_circleServer Pack Unlocker 설치
설치 전에는 허용 모드 전체 목록을 함께 확인해 주세요.
적용 순서: 위 - 특수광물 리소스팩
아래 - 띵타이쿤 온라인 공식 리소스팩
02

리소스팩 배치

띵팩 리소스팩이 공식 리소스팩보다 위에 오도록 배치합니다.

cloud_download자료실 열기

기본 배포본 / 추가 배포본 제공 중

03

옵션 설정과 점검

Continuity의 빛나는 텍스쳐 옵션을 켜짐으로 설정합니다.

auto_awesome
추가 안내

기존 구버전으로 선택해 사용할 수 있는 단일 발광 리소스팩도 함께 제공합니다.

문의 경로

문제 제보, 설치 문의, 개선 요청은 오른쪽 하단 채널톡으로 남겨 주세요.
